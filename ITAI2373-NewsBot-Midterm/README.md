# ITAI2373 NewsBot Midterm

## Project Overview
This project builds a NewsBot Intelligence System using Natural Language Processing (NLP). The system preprocesses news articles, extracts TF-IDF features, performs sentiment analysis, identifies named entities, and classifies articles into different news categories using machine learning. Multiple classification models, including Naive Bayes, Logistic Regression, and Support Vector Machine (SVM), were evaluated to identify the best-performing model.
